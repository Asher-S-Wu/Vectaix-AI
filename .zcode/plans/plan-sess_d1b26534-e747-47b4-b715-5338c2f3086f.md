# Vectaix AI 前端设计全面修复与美化计划

## 一、严重功能修复（4 项）

### 1. 修复首页引导卡片点击无反应
- `app/components/layout/ChatLayout.js`：渲染 `MessageList` 时传入 `onSendStarterPrompt={(text) => composerProps?.onSend?.({ text, attachments: [] })}`（复用现有发送管线 `handleSendFromComposer`）
- 顺带修复 `app/components/message/messageListUtils.js` 中引导语引号不配对问题

### 2. 修复移动端滚动锁泄漏到媒体页
- `app/globals.css`：移动端锁定滚动的规则改为 `html:not(.allow-page-scroll)` 条件下生效
- 新建一个极小的客户端组件（挂载时给 `<html>` 加 `allow-page-scroll` 类、卸载时移除），在 `app/media/layout.js` 中使用，让图片/视频页在手机上可以正常滚动

### 3. 修复触屏设备看不到消息操作按钮
- `app/components/message/MessageList.js`：给操作按钮容器加专用类名（如 `msg-actions`）
- `app/globals.css`：增加 `@media (hover: none)` 规则，触屏设备上这些按钮常显，不再依赖悬停

### 4. 恢复浮层阴影（美化重点）
- `app/globals.css`：删除末尾 `body * { box-shadow: none !important }` 全局禁用规则，以及暗色模式下已成死代码的 shadow 覆盖段
- 在 `:root` / `.dark-mode` 中定义一套阴影变量（浅色柔和投影、暗色更深投影），新建 `.shadow-pop` 工具类
- 给弹窗（AuthModal / ConfirmModal / ProfileModal / SystemPromptModal / UserManagementModal / ImageLightbox）、下拉菜单（ModelSelector / SettingsMenu / 导出菜单）、悬浮按钮（回到底部）统一应用 `.shadow-pop`，恢复层次感

## 二、暗色模式与主题一致性（3 项）

5. `app/globals.css`：聊天气泡暗色背景硬编码 `#1e1e22` 改用 CSS 变量，消除与周边面板的色差
6. `lib/client/hooks/useThemeMode.js` 和 `app/layout.js` 内联脚本中硬编码的 `#18181b` 统一改为 `#09090b`（与 `--bg-primary` 暗色值一致），消除橡皮筋滚动时的露底色差
7. 保留现有暗色 `!important` 覆盖体系（全站盲改风险太高），只修具体冲突点

## 三、死代码清理（1 项）

8. 移除未安装插件的无效类名：
   - `Markdown.js` 的 `prose-sm prose-zinc prose-pre:*`、`MessageList.js` 的 `prose-invert`（项目已有手写 `.prose` 样式）
   - `Composer.js`、`SystemPromptModal.js` 的 `animate-in fade-in slide-in-from-bottom-*`（这些组件本就有 framer-motion 或无需动画）
   - `Composer.js` 的 `scrollbar-none`，改为在 globals.css 写真实的隐藏滚动条 CSS

## 四、文案中文化与排版（4 项）

9. `Markdown.js` 代码块按钮 `COPY/COPIED` → `复制`/`已复制`；`FusionMessage.js` 徽标 `Fused` → `融合`；`MessageList.js` 导出菜单复用已有的中文 labelMap，不再显示大写英文
10. 工具调用标题的技术字符串回退（`tool.run` 这类）改为面向用户的中文表述；视频页 `${total} tokens` 等原始值改为中文文案
11. `app/layout.js`：`lang="en"` → `zh-CN`，metadata 描述改为中文；viewport `themeColor` 改为品牌白色 `#ffffff`
12. 全站省略号 `...` 统一为中文排版规范的 `…`

## 五、体验与视觉美化（5 项）

13. **图片/视频生成页**（`app/media/image/page.js`、`app/media/video/page.js`）：
    - 生成等待期间结果区显示骨架屏 + 加载动画（现在是一片空白）
    - 提交按钮、模式切换按钮补上 hover 效果
    - 错误文案不写死 "25MB"，改为从常量动态计算
14. **媒体工作台导航**（`app/media/layout.js`）：当前页高亮（图片生成/视频生成），头部加主题切换按钮（复用 `useThemeMode`）
15. **主按钮体系统一**：全站 4 套并存的主按钮配色收敛为一套 `bg-primary` 蓝色系；圆角收敛为卡片/面板 `rounded-2xl`、按钮 `rounded-xl`；消息操作按钮点击区域从约 26px 加大到 32px 以上
16. **图片查看器**（`ImageLightbox.js`）：监听窗口变化实时重算图片尺寸；内容图片补上中文替代文字（`alt="图片附件"` 等）
17. **杂项细节**：
    - `ChatHeader.js` 侧边栏按钮 aria-label 随开合状态变化
    - `ChatLayout.js` 移动端遮罩加半透明背景，有关闭提示感
    - `ConfirmModal.js` Enter 确认只在弹窗打开时响应，避免误触危险操作
    - `SystemPromptModal.js` `h-[85vh]` → `h-[85dvh]`，适配手机键盘弹出
